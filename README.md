## Sm_Warn
Sm_Warn is a warning plugin for csgo server with sourcemod installed.

**colorvariables.inc is need in order to run the script**

Usage for admins
```xml
sm_warn <username> "<reason_for_warning>"
or
/warn <username> "<reason_for_warning>"
```
